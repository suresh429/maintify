Maintify Account Deletion

If you would like to permanently delete your Maintify account and associated data, please send an email to:

ksuresh.unique@gmail.com

Subject: Delete Maintify Account

Please include:

  • Registered Email Address
  • Apartment Code
  • Apartment Name
  • Registered Mobile Number

What will be deleted?

  • User Profile
  • Login Credentials
  • Notification Tokens
  • Personal Account Information

What may be retained?

  • Apartment billing records required for apartment administration
  • Audit logs where required for security purposes

Deletion requests are processed within 7 working days.
